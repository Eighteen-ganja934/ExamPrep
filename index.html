Top 3 Most Likely Topics & Solutions

Topic 1

Bayes' Theorem

Common Pattern:

"A factory has 2/3 machines producing parts. X% are defective. A part is chosen randomly and is defective. Find the probability it came from machine Y."

Generic Solution Logic:

  1. Define Prior Events: Let E₁, E₂ be the machines. Define their probabilities P(E₁), P(E₂).
  2. Define Outcome Event: Let A = "Item is defective".
  3. Find Conditionals: Identify P(A|E₁) and P(A|E₂) from the text.
  4. Apply Formula:
    P(E₁|A) = [P(E₁)·P(A|E₁)] / [Σ P(Eᵢ)·P(A|Eᵢ)]
Topic 2

Normal Distribution

Common Pattern:

"Items are normally distributed with mean μ and variance σ². Find percentage of items having value > X or between X and Y."

Generic Solution Logic:

  1. Identify Parameters: Extract μ (mean) and σ (standard deviation). Careful: if variance is given, root it for σ.
  2. Standardize to Z: Convert the target value X to Z-score:
    Z = (X - μ) / σ
  3. Lookup & Calculate: Use the provided Z-table values (e.g., P(0<z<1)=0.3413) to find the area under the curve. Remember total area = 1, half = 0.5.
Topic 3

PDFs & Constant 'K'

Common Pattern:

"Given probability density function f(x,y) = K(x+y) for 0<x<1. Find K, and find P(X < 0.5)."

Generic Solution Logic:

  1. The Core Axiom: The total probability over the entire domain must equal exactly 1.
  2. Integrate/Sum: Set up the double integral (for joint continuous) or single integral:
    ∫ ∫ f(x,y) dx dy = 1
  3. Solve for K: Evaluate the integral with the given limits, equate to 1, and isolate K algebraically.

Process Flow: Solving Bayes' Theorem (Exam Strategy)

Step 1
Identify all 'Base' Events (e.g. Machines A, B) & calculate P(Eᵢ)
➜
Step 2
Identify the 'Observed' Event (e.g. Defective Item)
➜
Step 3
Extract Conditional Probabilities P(Observed|Base)
➜
Step 4
Multiply branches & Divide specific branch by total sum.

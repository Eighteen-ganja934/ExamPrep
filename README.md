## 🧰 Troubleshooting

- **The app does not open:** Try restarting your computer and run ExamPrep again.  
- **Installation fails:** Ensure you have permission to install software on your PC. Run the setup as administrator by right-clicking and selecting “Run as administrator.”  
- **Missing content:** Make sure you are connected to the internet on first launch to load the latest data.  
- **Crashes or freezes:** Check your system meets requirements and your Windows is updated.
